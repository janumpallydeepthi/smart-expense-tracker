# smart-expense-tracker
A full-stack expense tracking web app to manage daily spending and budgets.

## Features
- Add new expenses
- Categorize expenses (Food, Travel, Shopping, etc.)
- View list of expenses
- Delete expenses
- Simple and clean user interface

## Tech Stack
Frontend:
- HTML
- CSS
- JavaScript

Backend (Planned):
- Node.js
- Express.js

Database (Planned):
- MongoDB
